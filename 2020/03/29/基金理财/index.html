基金理财

发表于 2020-03-29 | 分类于 理财

债券基金

纯债基金

  • 纯债基金与非纯债基金，力哥建议投资优先考虑纯债基金，在想要投资债券基金的情况下。
    • 这里要注意的是，纯债基金中是允许投资可转债的，这种从严格意义上来讲是非纯债基金，如果投资范畴定位在百分之百纯债基金的话，那么要抛弃包含可转债的纯债基金。

投资技巧

  1. 基金基于四四三三的情况下；或者直接在天天基金网选择长时间的范围来排名前三十位的，其网站有长期纯债可选。
  2. 剔除投资了可转债的基金。
  3. 剔除小公司基金。
  4. 剔除基金规模太大的，比如超过20亿。
  5. 剔除同一家大型基金公司的非王牌基金。
  6. 挑选出综合费率最低的。
  7. 选择成立时间相对更长的，基金经理年龄较大，其管理时间较长的。

债基收费模式

那种类型更加合算：

  1. 如果只是想试试水，投资几个月也许就取出来了，那么选择不收取申购/赎回费用的模式更好。
  2. 如果想长期投资，那么可以采取申购/赎回的模式。

现在无论是前后端收费，费率听说都被压到很低了，所以前后端也许没有什么区别了，要具体比对。

定期开放式债券基金（定开债）

  • 定开债相对于普通债基，其封闭阶段，可以让基金经理更有操作空间。
  • 定开债，有可能存在更高杠杠率的情况（普通债基杠杠率不能超过140%，定开债是200%），通过债券A低压给银行，再去申购债券B这种模式，使得风险性增加。

    要注意的是，要追求更低风险的情况下，定开债依然要秉承着只选择纯债基的原则，这也是选择债基的原则。

交易所市场和银行间市场

  • 交易所市场类似股票的二级市场，但是市场流动性和股市不在同一个级别；
  • 银行间市场虽然流动性相对高，但是投资门槛高，老百姓玩不起；

混合型基金

  • 偏股混合型基金
  • 平衡混合型基金
  • 灵活配置混合型基金
  • 偏债混合型基金

    平衡和灵活配置才是混合型基金的精髓，因为可以做到风险对冲

股票型基金

重要指标

股票配置比例必须高于80%；

投资技巧

拼爹

  • 尽量选择大型的、知名的基金公司

    拼人

  • 基金经理，可以有三个考察维度：
    • 基金经理的基本背景（年龄、学历、履历）
      • 年龄建议至少在35岁以上；
      • 学历当然无上限；
      • 履历应该选择掌控对应那只基金年份更加久远的；
    • 操盘风格（短线、长线、基金偏好主题）
      • 基金每个季度都会公布一份基金报告；（在基金公司官网或者天天基金网的基金股票持仓报告中可以看出；或者在天天基金网里面的可以直接查看基金换手率， 不过这个一般每半年才公布一次，换手率50%~200%之间算正常的，超过200%算高的，说明是短线，换手率太高的要小心）

基金偏好风格

基金偏好风格依然可以在天天基金网查到相关信息；
【晨星投资风格箱】（在晨星中国的官网上可以查到所有股票型基金和混合型基金的风格箱，在天天基金也可以查到）

拼业绩

  • 基金成立的时间要足够久
    • 挑选成立三年以上的，最好是五年以上的
  • 投资领域的二手货往往比一手货更厚投资价值
  • 基金的长期业绩排名波动不能太大

晨星风格箱

Common Basic Knowledge

四四三三法则

基金分红

基金的分红与股票的分红是截然不同的

  • 基金不分红留在基金里面，净值不降低，钱依然是申购者的；
  • 股票不分红，那就是对应上市公司自家的，和股民无关；

    基金分红方式

  • 现金分红；
  • 红利在投资；

    基金的重要指标

  1. 累计净值（平时一般说的净值是单位净值，就是把历次所有分红收益都剔除掉之后当下的净值；一些新基金可能单位净值和累计净值一样，但是老基金分红多次之后，两个值可能相差很大）
  2. 单位净值（也就是净值，单位净值=（总资产－总负债）/基金总份额）
  3. 分红与否、基金单位净值高低与基金好坏没有必然联系

    负相关

    理论上以下关系较大概率是负相关（不排除出现正相关的情况）：
  • 股票–债券
  • 股票–房子
  • 存款–债券
  • 美元–黄金
  • 美元–石油
  • 美元–日元
  • 美元–比特币

    最专业最中立的三家基金评级机构

    基金业绩评价基准

  • 基金KPI
    • 根据基金自身特性定制的考核标准
  • β系数 = 1 代表基金真实业绩表现和业绩比较基准重合（β系数是对一个基金所承受的系统性风险的量化表达，可以理解为收益弹性有多大）
    • β系统越高，基金随市场涨跌幅度越大，反之越小

基金公司

只有红框的七家被评为五星公司

利率

利率=利息÷本金÷时间×100%，如果说年利率的话，就是将公式的时间换成一年
